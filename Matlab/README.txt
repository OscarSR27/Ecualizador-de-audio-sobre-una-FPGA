Instrucciones para el uso de los programas de Matlab

1. Utilice el programa Senoidal.m o Rampa.m para generar un vector de entrada a los filtros.

2. Utilice el programa DecimalAPuntoFijo.m para obtener el vector de entrada en formato binario punto 
   fijo. Debe ingresar el nombre del archivo que desea cargar. La salida del programa será el archivo
   Estimulo.txt	

3. Utilice el programa PuntoFijoADecimal.m para obtener en base 10 los resultados binarios a la salida
   de los filtros implementados en lenguaje Verilog. Debe ingresar el nombre del archivo que desea 
   cargar. La salida del programa será el archivo Salida.txt
